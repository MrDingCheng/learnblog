# C语言操作符

## 算术操作符

```
    +   -   *   /    %
```

## 移位操作符

```
    << 左移操作符
    移位规则：左边抛弃，右边补0
    >> 右移操作符
    移位规则：
    1.逻辑移位 左边用0补，右边丢弃
    2.算术移位 左边用原该值的符号位补充，右边丢弃
```

## 位操作符

```
    & 按位与
    | 按位或
    ^ 按位异或
    *他们的操作数必须是整数
```

## 赋值操作符

```
    =
    +=
    *=
    /=
    %=
    >>=
    <<=
    &=
    |=
    ^= 
```

## 单目操作符

```
    ！     逻辑反
    -      负值
    +      正值
    &      取地址
    sizeof 操作数的类型长度（以字节为单位）
    ~      对一个二进制数按位取反
    --     前置、后置--
    ++     前置、后置++    
    *      简介访问（解引用操作符）
    (类型)  强制类型转换
```
