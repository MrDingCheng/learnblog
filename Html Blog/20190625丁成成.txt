一、学习内容总结
1.a:链接
<a href=”index.html” target=”_blank”>链接元素</a>
href:链接地址     不可缺省
target="_blank"   打开链接目标窗口
        _blank    新的空白窗口下打开链接
        _self     在自己的窗口打开链接 
2.本页面锚点链接
a)定义锚点
	<a name=”mybottom”>元素</a>
b)链接锚点
	<a href=”#mybottom”>链接元素</a>
3.跨页面锚点链接
a)页面定义锚点
	<a name=”mybottom”>元素</a>
b)页面链接锚点,在链接上加一个页面地址
	<a href=”home.html#mybottom”>链接元素</a>
4.其他的行标签
a)	<span>元素</span>:范围
b)	<i>元素</i>:斜体
c)	<strong>元素</strong>:加粗
d)	<b>元素</b>:加粗
e)	<sub>元素</sub>:下标 
f)	<sup>元素</sup>:上标
5.html实体
a)	&nbsp;空格
b)	&lt;小于号<
c)	&gt;大于号>
d)	&copy;版权
e)	&reg; 注册商标
6.CSS(Cascading Style Sheet):层叠样式表
作用：
a)  美化界面
b)	在一定制度上解决了浏览器兼容性的问题
7.选择器
a)标签(元素、内联)选择器:
选择器名称与标签同名，同名标签将会适应此样式效果
定义格式：
选择器名{
   属性:属性值;
属性:属性值;
属性:属性值;
……
}
b)Class选择器:
定义时前面加.自定义名称。元素如果要适应此样式，需要给元素添加class属性
定义格式：
选择器名{
属性:属性值;
属性:属性值;
属性:属性值;
……
}
c)ID选择器:
定义前面加#选择器名称与某一个元素的id值同值，这个元素就会适应这个样式
定义格式：
#选择器名{
属性:属性值;
属性:属性值;
属性:属性值;
……
}
d)伪类选择器
定义格式：
a:link{
	color: red;
}
a:hover{
	color: green;
}
a:active{
	color: blueviolet;
}
a:visited{
	color: #FFA500;
}
其中的
:link     未被访问状态
:hover    鼠标悬停状态
:acitve   激活状态
:visited  访问过后状态
